# Sustavy RSYA — Design

**Дата:** 2026-06-24
**Цель:** Создать новый лендинг по восстановлению суставов и позвоночника (спикер Данила Сусак, центр Ksamata) как структурную копию лендинга `tirol/a`, по ТЗ «Подписная Ксамата» и контенту исходника `https://t.sustavy-spina.ru/spb`.

## Контекст

В `Lends/` живут отдельные лендинги, каждый — своя папка + свой git-репозиторий + свой Vite base-путь (поли-репо, НЕ монорепо):

- `tirol_rsya` → repo `landings-tirol-rsya-a`, base `/rsya/tirol/a/` (щитовидка)
- `tirol_rsya_b` → repo `landings-tirol-rsya-b`, base `/rsya/tirol/b/`

Шаблон лендинга: Vite, `index.html` + `src/main.js` (логика виджетов/каруселей/попапов) + `src/hero.js` (рендер секций строками) + `src/styles.css` + `tests/hero.test.js` (vitest). Виджет регистрации — GetCourse (inline + popup), аналитика — Яндекс.Метрика.

## Размещение

- Новая папка `Lends/sustavy_rsya/` — копия `tirol_rsya` (полной версии «a», на которую ссылается ТЗ).
- `vite.config.js` → `base: '/rsya/sustavy/a/'`
- `package.json` → `name: 'sustavy_rsya'`, `description: 'Landing page for Sustavy RSYA campaign'`
- Локальный `git init` (свой репозиторий `landings-sustavy-rsya-a` заведём позже).
- Существующий `tirol_rsya` (/rsya/tirol/a/) НЕ трогаем.

## Визуальное направление

Заменяем «ягодную/красную» палитру шаблона на **зелёную** (как у исходника t.sustavy-spina.ru): шалфейно-зелёные акценты, зелёные градиентные CTA-кнопки, тёплый светлый фон. Сохраняем из шаблона: типографику, стеклянные карточки, тонкие рамки, тени, адаптив (desktop/tablet/mobile).

Цвета вынесем в CSS-переменные (`--accent`, `--cta`, и т.п.), чтобы перекраска была точечной.

## Новые компоненты

### Program accordion (БЛОК 7)
- 5 раскрывающихся строк, стиль из image3: белая карточка, зелёный лейбл «ДЕНЬ N.», тема дня, круглая синяя/зелёная кнопка «+».
- Клик по строке раскрывает подпункты дня. Чистый JS в `main.js` (паттерн как у карусели/попапа), доступность (aria-expanded, кнопка).
- Данные дней — массив в `hero.js`, рендер строкой.

### Two-reasons (БЛОК 8)
- 2 карточки: круглое фото сверху + подпись. Центрированная CTA снизу.
- Адаптив: 2 колонки на desktop/tablet, стек на mobile.
